2. 메모리 사이클 피크 할인 검증

   시장의 실제 평가:
   
   MU:          Forward P/E 8.3 = 메모리 마진 압축 이미 반영됨
   SK하이닉스:  Forward P/E 4.8 = 극도 저평가 또는 함정
   
   향후 3년 마진 시나리오:
   
   시나리오             확률    주가 영향
   ─────────────────────────────────────
   마진 40-50% 유지      30%     +20%
   마진 30-35% 압축      50%     -15%  ← 가장 가능성 높음
   마진 25% 이하         20%     -40%
   
   → Risk-Reward: 불리함 (상승 기회 30%, 손실 리스크 70%)
   
   판정: 조건부 거부권 (마진 가이던스 확인 후 재평가)

4. DDOG P/E 42 (PEG 1.45) 할인 여부

   거부권 발동 — Risk Score 0.75 / 1.0
   판정: 신규 매수 금지
   
   | 지표 | DDOG | 평가 | 이유 |
   |------|------|------|------|
   | Forward P/E | 42.3 | 높음 | S&P평균의 1.9배 |
   | Trailing P/E | 360.5 | **극도** | 현재 이익 거의 없음 |
   | Operating Margin | 1.0% | **위험** | 매출 대비 이익 극소 |
   | PEG Ratio | 1.45 | 과평가 | PEG > 1.0 = 할인 거짓 |
   | Revenue Growth | +29.2% | 양호 | 성장은 좋으나... |
   | ROE | 3.3% | 극저 | 자본효율 최악 |
   
   핵심 문제:
   → 성장 29%는 좋으나, 마진 1%는 위험
   → Forward P/E 42를 정당화하려면 마진이 최소 10-15% 필요
   → 2023-2024 SaaS 대량 손실 사례 참조 (Okta -60% 등)
   → 고성장이 이익으로 전환되지 않는 기업 = 추가 성장 미충족 시 급락

KEY TAKEAWAYS

1. "저평가"의 함정: Forward P/E가 낮다 = 미래 이익 가정에 의존
   → 그 가정이 틀릴 가능성을 항상 평가할 것

2. CRWD 같은 고평가는 감정적 주장(전쟁 수혜)으로 숨겨짐
   → P/E 81 + 마진 음수 = 명백한 거부권 신호

3. 메모리 사이클(MU, SK)은 이미 할인 반영됨
   → 추가 수익 가능성 낮고, 손실 리스크는 높음

4. 데이터 기반 원칙:
   ✓ Operating Margin < 5% → 거부권 강화
   ✓ PEG > 1.2 → 과평가 신호
   ✓ Trailing P/E > Forward P/E 5배 → 미래 가정 의존도 높음
   ✓ P/E > 60 + 마진 < 10% → 즉시 거부
